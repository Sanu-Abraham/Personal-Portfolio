Veld — Design & Coding Agency Website

A modern, responsive website built from scratch for Veld, a design and development studio focused on crafting clean, high-performance digital experiences. The project explores the intersection of minimalism and visual depth, combining a dark interface with gradient-based lighting, glassmorphism, and subtle motion to create a premium, immersive feel. The site was developed using pure HTML, CSS, and JavaScript without relying on heavy frameworks, allowing for complete control over performance, structure, and interaction design.

A custom Three.js background was implemented to introduce a dynamic, interactive layer without compromising usability. Key features include a fully responsive layout, modular page architecture, interactive navigation, smooth scroll-based animations, and a consistent design system across all pages. Special attention was given to micro-interactions, typography, and spacing to maintain a refined and cohesive aesthetic. This project reflects a strong focus on frontend craftsmanship, performance optimization, and building scalable UI systems from the ground up.
